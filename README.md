一个简单的博客模板
========
###简介

看到一个老外的模板([戳这里看原版，需要翻墙][1])，但是细节欠缺，并且不支持移动端响应式布局。

自己照着它的样子，结合bootstrap仿出了主要的效果。

首页模板和分类模板应该很大程度上能重用，于是我就做了两个页面 首页和文章页。

文章页没有做评论部分，因为我打算采用第三方评论插件，如果你有兴趣，可以自己仿一下。

如果你觉得对你有用，请star一下 :)
